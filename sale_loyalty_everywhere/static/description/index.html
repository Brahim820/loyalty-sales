Loyalty Management for Website Sales and Back End Sales

This module allows to create Loyalty points for website and backend direct sales upon confirm sales. Loyalty will be automatically added and customer will have Loyalty Points for purchasing that product. Logic for Loyalty can be added in the configuration of the module in terms of each purchase or with amount spent.

Dependencies
  • sale.
Quick View
  • Create Loyalty for Specific product.
  • Create Loyalty for Specific category.
  • If no product or category selected, loyalty will apply to all products in all categories.
  • Add Loyalty on purchase.
  • Compute loyalty for customer.
  • Configurable Loyalty logic.

Overview and Uses

1. After installation go to Sale > Loyalty Programs > Loyalty Rule and hit 'Create'

2. Add items on Rules

3. Set loyalty name and hit save

4. To see uses Create sell order for that product

5. On Click for confirm sale it will automatically add loyalty
